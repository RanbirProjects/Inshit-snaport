## Operational Workflow

1.  **Reflection Input**: Capture raw professional observations regarding situational friction, decision-making, or performance variance.
2.  **Synthesis Engine**: Proprietary logic filters process the input to extract structural patterns and behavioral signals.
3.  **Output Report**: A high-fidelity executive summary including tactical reframing and professional liability checks.

## Deployment & Setup

1.  Clone repository.
2.  Ensure local environment has valid billing configuration.
3.  Install dependencies: `npm install`.
4.  Execute development build: `npm start`.

## Core Features

*   **Executive Logic**: Professional coaching persona focused on organizational ROI and career longevity.
*   **Persistence**: Secure local caching of session history.
*   **Privacy-First**: No data persistence outside the local client environment.
*   **Authentication**: Integrated billing profile selection for high-tier analytical processing.

## Security & Privacy

This application operates strictly within the client browser. No personal identifiable information is transmitted to external servers beyond the analytical processing endpoint. All sessions are ephemeral unless cached locally by the user.
